🏠 Real Estate Insights Dashboard (2011–2022)

An interactive Tableau dashboard built to analyze real estate transactions from 2011–2022, providing insights into sales trends, town performance, property distribution, and sales ratios.

The project demonstrates how data visualization can support municipal planning, taxation analysis, and housing development decisions through clear and interactive analytics.

📊 Project Overview

This dashboard enables stakeholders to explore the real estate market using interactive filters, highlight actions, and dynamic visualizations.

Users can analyze yearly sales trends, identify top-performing towns, and compare property types across regions.

🎯 Objectives

Analyze real estate transaction trends (2011–2022)

Identify top-performing towns based on sales

Understand property type distribution

Compare assessed value vs sale amount

Provide interactive insights for decision-making

📈 Dashboard Features

✔ Sales Trend Analysis – Line chart showing year-over-year sales trends

✔ Town Performance Analysis – Bar chart highlighting top towns by sales

✔ Property Type Distribution – Pie chart displaying residential, commercial, and industrial properties

✔ Residential Type Analysis – Treemap visualizing sales volume by residential category

✔ Geographical Sales Distribution – Map visualization showing town-level sales patterns

✔ Highlight Table – Comparison of average sales ratio by town and residential type

⚡ Interactive Dashboard Actions
